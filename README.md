## Controls

Right - Move right
Left - Move left
Down - Move down

S - Rotate left
D - Rotate right

R - Restart
Esc/Q - Exit

## Requirements

- Python 3.10
- [virtualenv](https://virtualenv.pypa.io/en/latest/)

## Setup

```bash
python3.10 -m virtualenv .venv
source .venv/bin/activate
pip install -r requirements.txt
python src/main.py
```
